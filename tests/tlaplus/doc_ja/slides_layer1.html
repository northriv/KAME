Layer 1: STM Commit

TLA+モデル解説 — stm_commit.tla (156行)

前提: Layer 0で検証済みのatomic_shared_ptrを正しいアトミックレジスタとして抽象化。 参照カウントの詳細はモデル化しない。

モデル化されるサイクル

Snapshot → Write → Commit(CAS) → 成功 or リトライ

変数 (7個) — 非常にシンプル

VARIABLES
    node_val,       \* 共有ノードのペイロード値
    node_serial,    \* シリアル番号（コミットごとに+1）
    pc,             \* "idle", "snapshot", "write", "commit", "done"
    thr_snap_val,   \* スナップショット時の値
    thr_snap_ser,   \* スナップショット時のシリアル
    thr_write_val,  \* 書き込む値
    thr_committed   \* 成功コミット回数

Layer 0の12変数に比べて大幅にシンプル。これが抽象化の威力。

iterate_commit サイクル

Step 1: Snapshot

TakeSnapshot(t) ==
    /\ pc[t] \in {"idle", "done"}
    /\ thr_snap_val' = [thr_snap_val EXCEPT ![t] = node_val]
    /\ thr_snap_ser' = [thr_snap_ser EXCEPT ![t] = node_serial]
    \* → atomic load 1回（Layer 0で検証済み）

Step 2: Write（2種類をモデル化）

(* 2a: 非決定的書き込み — 任意のクロージャ *)
Write(t) ==
    /\ \E v \in 0..MaxVal :
       thr_write_val' = [thr_write_val EXCEPT ![t] = v]

(* 2b: 決定的インクリメント — tr[node].x += 1 *)
WriteIncrement(t) ==
    /\ thr_write_val' = [thr_write_val EXCEPT
         ![t] = thr_snap_val[t] + 1]

Step 3: Commit (CAS)

Commit(t) ==
    \/ (* CAS成功: 値とシリアルの両方がスナップショットと一致 *)
       /\ node_val = thr_snap_val[t]
       /\ node_serial = thr_snap_ser[t]
       /\ node_val' = thr_write_val[t]       \* 新値をインストール
       /\ node_serial' = node_serial + 1     \* シリアル+1
       /\ thr_committed' = [thr_committed EXCEPT ![t] = @ + 1]

    \/ (* CAS失敗: 他スレッドがコミット済み → リトライ *)
       /\ ~(node_val = thr_snap_val[t] /\ node_serial = thr_snap_ser[t])
       /\ thr_snap_val' = [thr_snap_val EXCEPT ![t] = node_val]
       /\ thr_snap_ser' = [thr_snap_ser EXCEPT ![t] = node_serial]
       /\ pc' = [pc EXCEPT ![t] = "snapshot"]

ポイント: CAS失敗時、即座に新しいSnapshotを取ってリトライ。これがiterate_commitの動作。

検証された不変条件 (5件 + TypeOK)

\* 1. Lost Update なし: 2スレッドが両方コミット成功→シリアルは2以上増加
NoLostUpdate ==
    \A t1, t2 \in Threads :
        (t1 /= t2 /\ thr_committed[t1] > 0 /\ thr_committed[t2] > 0)
        => node_serial >= 2

\* 2. コミットは直列化: 全スレッドのコミット回数合計 ≤ シリアル
CommitSerializes ==
    totalCommits <= node_serial

\* 3. スナップショットはコミット前: コミッタのsnap_ser < 現在のserial
SnapshotBeforeCommit ==
    \A t \in Threads :
        (pc[t] = "done" /\ thr_committed[t] > 0)
        => thr_snap_ser[t] < node_serial

\* 4. 値は範囲内
ValueBounded == node_val <= MaxVal

\* 5. 最後のコミッタの書き込み値が見える
WriteReadConsistency ==
    \A t \in Threads :
        (pc[t] = "done" /\ thr_snap_ser[t] = node_serial - 1)
        => node_val = thr_write_val[t]
