説明

template<typename T>
class atomic_scoped_ptr< T >

This is an improved version of boost::scoped_ptr<>. atomic_scoped_ptr<> can be shared among threads by the use of swap() as the argument. That is, a destructive read. Use atomic_shared_ptr<> for non-destructive reading.
