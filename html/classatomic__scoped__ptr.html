説明

template<typename T>
class atomic_scoped_ptr< T >

This is improved version of boost::scoped_ptr<>. This object itself is not atomically changed. However, targeted object ( argument of swap() ) can be atomically swapped. Therefore, atomic_scoped_ptr<> can be shared among threads by the use of swap() as a argument. That is, destructive read. Use atomic_shared_ptr<> for non-destructive reading.
